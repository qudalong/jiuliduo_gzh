# 酒利多商城

#### 介绍

商城，单页面应用，主要开发技术栈为html5 css3 vue2+ vuecli3 vuex以及其他插件。

#### 项目结构


api           # API调用

assets        # 静态文件

components    # 封装组件

pages         # 页面 文件

request       # ajax调用，异常提示

routes        # 页面路由

service       # 设置

store         # vuex store reducer  effects 设置

#### 项目命令

启动：npm run serve

打包：npm run build
